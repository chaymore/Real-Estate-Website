# Jenny Haymore Real Estate Website

Welcome! This is the start of your mom's professional real estate website. Let me explain what each file does and how to use them.

## 📁 Files You Have

### 1. **index.html** (The Home Page)
This is the main page people see when they visit the website. It's written in HTML (HyperText Markup Language).

**What HTML does:** HTML creates the structure and content of the page - like the skeleton of a house. It defines headings, paragraphs, images, buttons, etc.

**Sections in this file:**
- Navigation bar at the top (links to other pages)
- Hero section (big header with a headline and headshot)
- Bio section (introduction to Jenny)
- Featured listings (3 example properties)
- Testimonials (client reviews)
- Contact form
- Footer (bottom of the page)

### 2. **styles.css** (The Styling)
This makes everything look beautiful! CSS stands for Cascading Style Sheets.

**What CSS does:** CSS is like the paint, furniture, and decorations of a house. It controls:
- Colors (we used soft neutrals: white, beige, and blue)
- Fonts and text sizes
- Spacing and layout
- Animations and hover effects

**Key design choices:**
- Primary blue: #5B8DB8 (soft, professional blue)
- Beige: #F5F1E8 (warm, inviting background)
- Clean, modern layout inspired by Brie Green's website

### 3. **script.js** (The Interactive Features)
This adds interactive behavior to the website. JS stands for JavaScript.

**What JavaScript does:** JavaScript is like the plumbing and electricity - it makes things work when you interact with them:
- Smooth scrolling when clicking navigation links
- Form submission handling
- Animations when scrolling down the page
- Navbar shadow effect when scrolling

## ✏️ How to Edit Content

### Changing Text:
1. Open `index.html` in a text editor (like Notepad on Windows or TextEdit on Mac)
2. Find the text you want to change (it's between HTML tags like `<p>` and `</p>`)
3. Edit it
4. Save the file
5. Refresh your browser to see the changes

**Example:**
```html
<h1>Find Your Dream Home</h1>
```
You can change "Find Your Dream Home" to anything you want!

### Changing Colors:
1. Open `styles.css`
2. Find the `:root` section at the top
3. Change the color codes (the # numbers)
4. Save and refresh

### Changing Contact Information:
Look for this section in `index.html`:
```html
<div class="contact-item">
    <h3>📞 Phone</h3>
    <p>(555) 123-4567</p>
</div>
```
Replace the phone number with Jenny's real number!
